How TruthLens Works

A complete breakdown of the AI pipeline, tools, and technology powering the detection system.

01 — Pipeline

System Workflow

Six stages from raw text to a verified prediction.

01
User Input
The user pastes a news article into the web interface, or the system fetches a live headline from NewsAPI (BBC News / Reuters).
02
⚙️
Text Preprocessing
The text is cleaned: converted to lowercase, URLs are removed, special characters and punctuation are stripped, and extra whitespace is normalized.
03
TF-IDF Vectorization
The cleaned text is transformed into a numerical feature vector using Term Frequency–Inverse Document Frequency (TF-IDF) with 50,000 features and bigrams.
vectorizer.transform([cleaned_text]) → sparse matrix (1 × 50,000)
04
🤖
ML Classification
The Passive Aggressive Classifier receives the feature vector and predicts a class: REAL or FAKE. It also returns a decision function score used to compute confidence.
model.predict(features) → "REAL" | "FAKE"
05
Confidence Calculation
The raw decision score is normalized to a 0–100% confidence value using a sigmoid-style transformation, giving users a human-readable probability.
06
✅
Result Display
The prediction, confidence score, flagged keywords, and word count are returned and displayed to the user. The result is also saved to the SQLite database.
02 — Machine Learning

The ML Model

Passive Aggressive Classifier

The PAC is an online learning algorithm well-suited for large-scale text classification. It stays "passive" when a prediction is correct, but makes aggressive updates when it makes a mistake — making it fast and highly accurate for news classification.

AlgorithmPassive Aggressive Classifier
RegularizationC = 0.5
Max iterations50
Test accuracy~96–99%
TF-IDF Vectorizer

TF-IDF (Term Frequency–Inverse Document Frequency) converts text to numerical vectors. Words that appear frequently in a document but rarely across all documents are weighted higher — identifying the most meaningful terms.

Max features50,000
N-gram range(1, 2) — unigrams + bigrams
Max document freq70%
Sublinear TFEnabled (log normalization)
Training Dataset

The model was trained on a public dataset of real and fake news articles sourced from Kaggle. The dataset contains articles from various topics including politics, world events, and social issues.

Total articles~44,956
Fake articles~23,481
Real articles~21,417
Train/Test split80% / 20%
